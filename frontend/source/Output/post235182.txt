Create custom Terminal shortcuts <body> I'm working a lot with SSH connections to servers, by for example forwarding localports or just establishing SSH connection to manage some servers.  For this I would like to be able to create some terminal shortcuts for example:  precodeconnect ssh server1 --&gt;should run command: ssh xyx@xy connect ssh server2 forward --&gt;should run command: ssh -NC user@xy -L 9999:localhost:3306 /code/pre  Is there a way to specify custom terminal commands?  <answer235183> Simply add aliases in ~/.bash_profile:  If the file .bash_profile doesn't exist:  precodetouch ~/.bash_profile /code/pre  Then add aliases with codenano ~/.bash_profile/code. Examples:  precodealias ssh01='ssh xyx@xy' alias ssh02='ssh -NC user@xy -L 9999:localhost:3306' /code/pre  Then source the file or restart Terminal.app:  precodesource ~/.bash_profile /code/pre  Entering codessh01/code in the shell will then execute codessh xyx@xy/code or codessh02/code the second command.  The alias (i.e. ssh01) mustn't be another valid command in your path (e.g ssh-add)  <comment287647> And if you want to define custom commands, not just aliases, just `chmod +x` an sh file (without extension) and put it in your `~/bin`.
