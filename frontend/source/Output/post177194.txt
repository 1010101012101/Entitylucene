For posterity, I thought I'd at least mention this to anyone else that there's an extra thing to do to install XCode.  <answer177196> Yes, you can install it in /Applications.  <answer177348> After downloading the Xcode.dmg, double-click the image to mount it (if it's not already mounted automatically).  You will find a pkg file inside. .pkg is the file extension for software packages in OS X (and iOS). If you double-click the package the related app "Installer" will open it and you may install it.  Later Xcode images (e.g. Xcode 4.6.3 - also for Lion) already contain an app bundle: Xcode.app. To "install" it just copy/drag it to the Applications folder.  <comment209495> do I have to "run" it after double-click install? <comment209498> If the installxcode_421_lion.dmg didn't automatically mount you double-click it to mount it then double-click the installer that's in the Finder window which opens from the .dmg file being mounted.
