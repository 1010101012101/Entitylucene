When I try to open it, it says "Application blocked for security reasons" because "The certificate has expired".  img src="https://i.stack.imgur.com/XCiT7.png" alt="Java Application Blocked dialog"  I really want to launch it... So, is there a way to open and use it?  I tried to go to the Java settings and add the link to the applet to the white list, but it does not work.  <answer244745> Just to be clear, the reason you are getting this error isn't because of a problem with Java on your Mac. The problem is with a certificate that has been used to sign one or more of the JAR files that make up the TopCoder Java application.  The certificate expired on March 25th this year.  img src="https://i.stack.imgur.com/oSFDq.png" alt="Certificate details for expired certficate"  If you still want to run the application, even though the certificate has expired, then you need to add it to the Exception Site List within Java Control Panel.  You may want to update Java on your Mac first though. You can get the latest download from Oracle here, Apple don't supply their own Java runtime anymore.  img src="https://i.stack.imgur.com/qR2cF.png" alt="Security tab of Java Control Panel"  ol liLaunch Java Control Panel from System Preferences/li liSelect the Security tab/li liClick on "Edit Site List..."/li liClick on "Add"/li liEnter "http://www.topcoder.com" in the Location field/li liClick OK/li liClick Apply/li /ol  Now when you try to open the JNLP file you'll get a different message. Just check the "I accept the risk and want to run this application" and click Run.  img src="https://i.stack.imgur.com/ZxnTK.png" alt="Do you want to run this application? Security Warning dialog"  At which point the application should successfully download and run.  img src="https://i.stack.imgur.com/dNqDg.png" alt="TopCoder application"  <comment301585> Have you got the latest java update from Apple? <comment301589> Yep, I think so.
