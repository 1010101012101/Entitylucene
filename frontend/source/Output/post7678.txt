How do I suppress the address bar in mobile Safari? <body> When I add the Nibbble web app to my iPhone home page, it not only installs an icon, but when run, also hides all of the mobile Safari chrome.   How does it achieve this?  <answer7683> You only need the first part of this code to do what you ask, but the rest may also be helpful.  precode// hide the browser chrome &lt;meta name="apple-mobile-web-app-capable" content="yes" /&gt;   //set the phone status bar style; can be grey, black, or black translucent &lt;meta name="apple-mobile-web-app-status-bar-style" content="black" /&gt;  // set the viewport for iOS &lt;meta content="minimum-scale=1.0, width=device-width, maximum-scale=0.6667, user-scalable=no" name="viewport" /&gt;  // set a custom icon for when a user bookmarks the app to the home screen &lt;link rel="apple-touch-icon" href="/path/to/icon.png" /&gt; /code/pre  <comment8323> Good answer from @zevlag but this probably belongs on StackOverflow.com <comment8324> Good answer! +1 <comment124143> Why is ```maximum-scale=0.6667``` ?
