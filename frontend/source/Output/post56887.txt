Mencoder on Lion <body> Is there a way to install codemencoder/code on Lion, preferrably using homebrew?  <answer56892> codemencoder/code is part of the MPlayer utility, and is installed as part of the MPlayer package on most systems. Installing MPlayer via Homebrew ( codebrew install mplayer/code ) will supply the codemencoder/code binary as well.  precode$ brew list mplayer /usr/local/Cellar/mplayer/1.1/bin/mplayer /usr/local/Cellar/mplayer/1.1/bin/mencoder /usr/local/Cellar/mplayer/1.1/share/man/ (2 files) /code/pre
