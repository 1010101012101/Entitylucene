How to change color of TODAY in Calendar? <body> How can I change the color of TODAY in Calender?  <answer189619> In OS X it’s possible to change the highlight color. You can do this in the General ▸ Appearance panel in System Preferences.   Now you can enable the system highlight color for the today selection with the following command.  precodedefaults write com.apple.ical CalUIUseSystemHighlightColorForToday /code/pre
