In order to boot to Bootcamp EFI partition, you have to codebless/code the codeEFI/code partition of the disk that you install Bootcamp, too. So it goes something like this: codebless -device /dev/disk0s1 -mount /Volumes/BOOTCAMP -setBoot -nextonly/code.  <comment134948> I know this isn't via command line, but for testing purposes does [BootChamp](http://www.kainjow.com) work? <comment134949> BootChamp sounds like a BootCamp-oriented software.
