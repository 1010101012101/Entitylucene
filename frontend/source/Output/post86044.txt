How to install g++-45 with Homebrew <body> I have installed gcc45 by using codebrew tap homebrew/versions &amp; brew install gcc45/code. However, I can not find g++-4.5 in code/usr/local/bin/code. How can I install g++-4.5 separately by Homebrew? Thanks.  My system 10.6.8.  <answer89758> Looks like the g++ part is not built by default. Build it with codebrew install gcc45 --enable-cxx/code and codeg++-4.5/code will appear.
