Osascript Compile Application <body> Is there a way to compile an an applescript from the command line similar to the following. img src="https://i.stack.imgur.com/hrBOA.png" alt="enter image description here"  <answer171691> The answer to this is surprisingly simple:  precode$ osacompile -o output.app input.applescript /code/pre  This works for both plaintext code.applescript/code files and the code.scpt/code files that Script Editor makes.  You can use a pipe too, e.g. if you have the script on your clipboard:  precode$ pbpaste | osacompile -o output.app /code/pre  Or code-e/code, like the codeosascript/code command can use:  precode$ osacompile -o output.app -e "$(pbpaste)" /code/pre  You can also use JavaScript with the code-l/code flag:  precode$ osacompile -o output.app -l JavaScript -e 'app = Application.currentApplication();' -e 'app.includeStandardAdditions = true;' -e 'app.displayDialog("hi");' /code/pre
