Getting Applications location in Automator <body> How do I get the directory where the app created in automator resides?  Example:  ul liCreate App using automator/li liSave to ~/Documents/Automator/li liRun automator app, which does something in it's directory/li /ul  <answer23838> Add an AppleScript step which contains:  precodeon run {input, parameters}      set p to POSIX path of (path to me)     return {p}      return input end run /code/pre  From there, you can use the "p" parameter.
