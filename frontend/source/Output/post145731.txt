How to trigger Notification Centre notification from JavaScript? <body> The other question asks the same but using AppleScript. Apple introduced JavaScript as a new language for Automation in OS X 10.10.  <answer145732> From Terminal,  precodeosascript -l JavaScript -e "app=Application.currentApplication(); app.includeStandardAdditions = true; app.displayNotification('Hello, world', {withTitle: 'Title'})" /code/pre  From Script Editor,  precodeapp = Application.currentApplication() app.includeStandardAdditions = true app.displayNotification('Hello, world', {withTitle: 'Title'}) /code/pre  <comment171756> We need to link to the API documentation on the displayNotification function; I don't know what other parameters it accepts. <comment318424> The documentation is available in Script Editor's Library under StandardAdditions / User Interaction / displayNotification. The other available options are `subtitle` and `soundName`.
