Directory /cores in Mountain Lion <body> Is the directory /cores in OSX Mountain Lion important? It's empty on my machine and has been for some time, so I was considering deleting it, but I'd like to not mess anything up.  <answer61166> I suggest not deleting any Unix folders on the drive if you not wish to reinstall the OS. It might be that any function trying to access it will create it if not there but might also end in problems and errors.  <answer61176> The /cores directory is usually hidden and is used for debug information after crashes. Any files inside are probably safe to delete but it's generally a bad idea to remove any system directories. It is normal and should be present on all OS X installs.  <answer108072> The directory fills with core dumps from programs that crash. You can freely delete any file there. Mine routinely fills up to tens of gigabytes and it is never pruned by Apple's OS.
