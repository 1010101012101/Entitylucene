How to turn off the iOS-style character picker menu <body> When I hold a key in Lion, I get this iOS-style character picker instead of a repeated character, as in previous versions of the OS:  img src="https://i.stack.imgur.com/RVtbM.png" alt="screenshot"  I really dislike this (I rarely use those symbols as I only speak Dutch and English). I want to hooooooooooold a key. :)  Where can I turn this off? I can't find an option in the system preferences.  <answer20619> You can do it with Terminal:  http://m10lmac.blogspot.com/2011/07/os-x-107-lion-getting-rid-of-character.html  Extract:       A new Lion feature taken over from iOS is the "Character Picker", which generates a popup menu of accented characters when you hold down the base letter on the keyboard. Some users detest this, as it stops the key repeat function which normally results when you hold down a key. To get rid of Character Picker, open Terminal and type      codedefaults write -g ApplePressAndHoldEnabled -bool false/code      and then restart.   <answer20620> You have to open a Terminal window and type in the following:  precodedefaults write -g ApplePressAndHoldEnabled -bool false /code/pre  (from this article)  <answer20682> If you’re interested in more of these "secret preferences" check out the awesome Secrets website.
