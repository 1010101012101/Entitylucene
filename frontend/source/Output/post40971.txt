Is there a way to globally change the appearance of messages in Messages (Beta) for OS X? <body> In OS X Messages (Beta), the appearance of a given message thread can be changed by selecting submenu options from the codeView &gt; Messages/code menu item; but this only applies to the currently displayed message. Is there a way to configure these options as defaults for (all) new threads?  <answer44545> Yes, there is:  precodeMessages &gt; Preferences &gt; Messages /code/pre  You can set the look and feel for incoming and outgoing messages here.  img src="https://i.stack.imgur.com/K4jmh.png" alt="enter image description here"
