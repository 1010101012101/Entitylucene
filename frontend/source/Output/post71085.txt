What terminal command or program can list an SSD's Power On Hours? <body> I'm looking to script using the unix shell the retrieval of SSD (and HDD) data that disk utility shows in the information window.  img src="https://i.stack.imgur.com/9n2IX.png" alt="Disk Utility information"  I'm immediately interested in the Power On Hours and Power Cycle, but if all the data comes from one data structure or tool, so much the better. The usual tools codediskutil info disk0/code and codeioreg -l/code are not producing the "of interest" values listed in the screen shot above.  Is this information available from any of the built in OS X terminal tools or do I need to locate an open-source package that makes the same API calls to get this level of detail about a Mac drive?  <answer73216> codesmartmontools/code is an open source package that provides most of the values you are interested in.  Download it from http://sourceforge.net/apps/trac/smartmontools/wiki/Download and compile it as usual:  precode$ cd smartmontools-6.0 $ ./configure  $ make $ sudo make install /code/pre  (If you prefer, install it with MacPorts or Fink.)  Unless otherwise specified as an option to code./configure/code, codesmartmontools/code commands install in code/usr/local/sbin/code, so update your code$PATH/code variable if necessary.   The tool we are interested in is called codesmartctl/code.   First activate S.M.A.R.T.:  precode$ /usr/local/sbin/smartctl /dev/disk0 -s on /code/pre  Then run it:  precode$ /usr/local/sbin/smartctl /dev/disk0 -x /code/pre  This is an excerpt of the output, which provides a wealth of information.
