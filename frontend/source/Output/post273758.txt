Is there a simple way to convert a Unicode text file to PDF on the command line on macOS? <body> What is a simple way to convert a Unicode text file to PDF on the command line on macOS?  One can open a text file in e.g. TextEdit and print it as a PDF. Is there a command line equivalent?  <answer273759> The following has been tested on Mac OS 10.12.1.  To convert a Unicode text file text.txt to a pdf file text.pdf:  precodetextutil -convert html test.txt cupsfilter test.html &gt; test.pdf /code/pre  To specify font:  precodetextutil -font 'Menlo Regular' -fontsize 11 -convert html test.txt cupsfilter test.html &gt; test.pdf /code/pre
