Only Macintosh shown on Startup Manager after Windows 10 32-bit Boot Camp install <body> I installed Boot Camp with Windows 10 32-bit, but it doesnt give me the option to switch my Mac into Windows. Any help? I've tried holding option when restarting it but only the Macintosh box appears.  <answer267101> 32-bit versions of Windows 10 are not supported on any Mac model. You need to install the 64-bit version.
