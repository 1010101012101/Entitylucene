Modify pwpolicy in Sierra <body> I am trying to modify default Sierra password policies through codepwpolicy/code, trying for example, to lower the depth of password history from 24 to 1 by hitting as sudoer:  precodesudo pwpolicy setaccountpolicies "policyAttributePasswordHistoryDepth=1" /code/pre  But it is not working. The error message is:     Error: Policy file  not found   The policyIdentifier I try to change that gets listed (with many more) when hitting codepwpolicy getaccountpolicies/code reads like this:  precode&lt;key&gt;policyIdentifier&lt;/key&gt;              &lt;string&gt;ProfilePayload:[HASHED_VALUE]:pinHistory&lt;/string&gt;            &lt;key&gt;policyParameters&lt;/key&gt;              &lt;dict&gt;   &lt;key&gt;policyAttributePasswordHistoryDepth&lt;/key&gt;   &lt;integer&gt;24&lt;/integer&gt;              &lt;/dict&gt; /code/pre  How is the proper way of doing it?  <answer293879> Ok, this is how I finally resolved it, though I am not sure if it is the properest way.  As @patrix pointed out, codepwpolicy setaccountpolicies/code requires for the path of a XML file with the policies.  codepwpolicy getaccountpolicies/code gives the current global policies in XML format, so I redirected the output to a file with codepwpolicy getaccountpolicies &gt; file.plist/code , removing the possible info messages the command might write and leaving only the XML structure.  precode&lt;?xml version="1.0" encoding="UTF-8"?&gt; &lt;!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd"&gt; &lt;plist version="1.0"&gt; &lt;dict&gt;     &lt;key&gt;policyCategoryPasswordChange&lt;/key&gt;     &lt;array&gt;     ...     &lt;/array&gt; &lt;/dict&gt; &lt;/plist&gt; /code/pre  Then I changed the value I wanted to modify in my case, in the file I created (codefile.plist/code):  precode&lt;key&gt;policyIdentifier&lt;/key&gt;              &lt;string&gt;ProfilePayload:[HASHED_VALUE]:pinHistory&lt;/string&gt;            &lt;key&gt;policyParameters&lt;/key&gt;              &lt;dict&gt;   &lt;key&gt;policyAttributePasswordHistoryDepth&lt;/key&gt;   &lt;integer&gt;2&lt;/integer&gt;              &lt;/dict&gt; /code/pre  Finally I used that file to update the policy: codepwpolicy getaccountpolicies /path/to/file.plist/code  <comment370802> I've never used the command, but according to the man page `-setaccountpolicies` "Takes one                                   argument: the name of the file containing the policies." <comment370803> I can't find the file nor the path for the XML list of policies.
