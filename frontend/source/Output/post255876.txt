In the past, the same mobile configuration worked well with previous versions of OS/X.  Even more puzzling, when feeding the mobile profile, system preferences open however, neither it does not configure the VPN, nor it displays any error.  The VPN itself is using Cisco IPsec configurations, and is well supported.
