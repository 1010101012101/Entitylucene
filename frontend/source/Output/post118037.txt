What should I do to fix this issue?  <answer118039> There are two possibilities. The font could be "disabled" or there could be a problem with the font files.  Run codeFont Book/code and have a look to see if the font is marked as codeOff/code in either your computer or user list.  Then highlight the font in the font list and select codeFile/code  codeValidate Font/code and the font will be checked. This is quite a swift process so I usually check them all. A dialog will open that lists all the fonts and shows you the ones that may have a problem. The most common (and easily ignored) warning is that you have a duplicate font but if you have a duplicate of one you are having a problem with I'd remove the 'User' version.  If that doesn't work then you might try codeFile/code  codeRestore Standard Fonts.../code which moves all the non-standard fonts out of the way into a folder called codeFonts (Removed)/code next to the codeFonts/code folder.  <answer118057> Try installing Helevetica again. I remember i had some font issues a few years ago, I would download the font again and it would be working.  <answer118058> Actually you emdon't/em have the correct version of Helvetica in the right spot.  You are missing the files codeHelvetica.dfont/code and codeHelveticaNeue.dfont/code which are the correct files for the missing fonts. They should be in /System/Library/Fonts. Have a good look at the support page you point to and you will see those two files listed against the fonts.  You do have two files, codeHelvetica.ttc/code and codeHelveticaNeue.ttc/code, which are strongnot/strong included in a standard Mavericks install but they are not where the System is expecting to find the two fonts.  As an experiment move these two files into ~/Library/Fonts and see what happens (I'd be interested to know the result).  In the long term grab a copy of the two missing files from another Mac running 10.8 or 10.9 and put them into code/System/Library/Fonts/code. You will have to do this by hand since Font Book will only install fonts in either code/Library/Fonts/code or code~/Library/Fonts/code.  <comment137772> Is Helvetica listed in Font Book  ? <comment137782> If the fonts were merely disabled, they would still appear in /Library/Fonts.
