What's its usage?  <answer130924> There is a command line wireless utility called codeairport/code, but it is not in the user $PATH to be able to call it without remembering where it is.   precode/System/Library/PrivateFrameworks/Apple80211.framework/Versions/Current/Resources/airport /code/pre  so to make it available at the line, and put it where it really belongs, in Terminal:  precode chillin@stack:~$   chillin@stack:~$   chillin@stack:~$ sudo ln -s /System/Library/PrivateFrameworks/Apple80211.framework/Versions/Current/Resources/airport /usr/sbin/airport  chillin@stack:~$  /code/pre  And when calling it, gives great info about the wireless network you're connected to:  precode chillin@stack:~$   chillin@stack:~$   chillin@stack:~$ airport -I       agrCtlRSSI: -53       agrExtRSSI: 0      agrCtlNoise: -84      agrExtNoise: 0            state: running          op mode: station       lastTxRate: 48          maxRate: 54  lastAssocStatus: 0      802.11 auth: open        link auth: none            BSSID: c0:a7:50:5d:46:b9             SSID: starbucks              MCS: -1          channel: 3  chillin@stack:~$   chillin@stack:~$   chillin@stack:~$ # ok, what else?
