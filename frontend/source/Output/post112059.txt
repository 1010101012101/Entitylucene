But I would like those to run as "services", i.e., not as part of a User account, and without having to log in.  <answer116827> You can use stronglaunchd/strong, tmux and this script to launch rTorrent as a daemon.  This post describes how to install rTorrent and run it as a daemon using tmux. First, install software:  precode$brew install rtorrent --with-xmlrpc-c $brew install tmux /code/pre  Parameter â€“with-xmlrpc-c is needed to use GUI clients with rTorrent. rTorrent needs terminal to run. We provide virtual terminal using tmux. To autostart rTorrent create launchd sript and place it to  precode/Library/LaunchAgents/rtorrent.plist /code/pre  The script:  precode&lt;?xml version="1.0" encoding="UTF-8"?&gt; &lt;!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd"&gt; &lt;plist version="1.0"&gt; &lt;dict&gt;     &lt;key&gt;Label&lt;/key&gt;     &lt;string&gt;rtorrent&lt;/string&gt;     &lt;key&gt;ProgramArguments&lt;/key&gt;     &lt;array&gt;         &lt;string&gt;/usr/local/bin/tmux&lt;/string&gt;         &lt;string&gt;new-session&lt;/string&gt;         &lt;string&gt;-s&lt;/string&gt;         &lt;string&gt;rtorrent&lt;/string&gt;         &lt;string&gt;-d&lt;/string&gt;         &lt;string&gt;rtorrent&lt;/string&gt;     &lt;/array&gt;     &lt;key&gt;KeepAlive&lt;/key&gt;     &lt;true/&gt; &lt;/dict&gt; &lt;/plist&gt; /code/pre  Load script:  precode$sudo launchctl load -w /Library/LaunchAgents/rtorrent.plist $launchctl list | grep rtorrent /code/pre  If all is ok you will see something like this:  precode- 1 rtorrent 4868 - 0x7f872252ff90.anonymous.rtorrent /code/pre  Now you can attach to tmux session and start using rTorrent:  precode$tmux attach-session -t rtorrent /code/pre  To restart script unload it and load again:  precode$sudo launchctl unload -w /Library/LaunchAgents/rtorrent.plist $sudo launchctl load -w /Library/LaunchAgents/rtorrent.plist /code/pre  To view launchd logs run:  precode$sudo launchctl log level debug $tail -f /var/log/system.log | grep launchd /code/pre  The rTorrent installation is over, now you can install GUI client.
