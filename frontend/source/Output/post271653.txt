External HD can be fully accessed only via the terminal but not the finder <body> Currently, I use this script to mount my external HD(ntfs)  precode  1 #!/bin/bash   2 extNameKeyword="TOSHIBA"   3   4 devPath=/dev/$(diskutil list | grep $extNameKeyword | awk 'NF&gt;1{print $NF}')   5 mountPoint="/Volumes/manual_ext"   6   7 mkdir $mountPoint   8 sudo /usr/local/bin/ntfs-3g -o uid=502,gid=20 $devPath $mountPoint -olocal -oal    low_other   9  10 exit 0 /code/pre  The result is I can get a full read-write-execute permission only when I access the drive via a terminal.
