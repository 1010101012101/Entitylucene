I don't want to run my MacBook Pro as a server. I ran this app just to test something, now I want to reset to the previous state. Now I feel that there are some daemon process running because of the server app, that slows down my laptop. those directories to uninstall Server I was getting this error when running `sendmail`:  `sendmail: fatal: open /Library/Server/Mail/Config/postfix/main.cf: No such file or directory` To fix this, edit `/etc/postfix/main.cf` and remove the line `config_directory = /Library/Server/Mail/Config/postfix`. Or change it back to `server_config_directory = /etc/postfix`. Then it'll use the configs in `/etc/postfix` again instead of the now non-existent configs from OSX Server. <comment284932> Your /Library/Keychains.keychain item in the replaced list, should be /Library/Keychains/System.keychain as Server adds several identities & password items to the System keychain.
