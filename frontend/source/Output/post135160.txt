How can I list only interface name and assigned IP with ifconfig in terminal <body> Is there a way using only codeifconfig/code to list strongonly/strong network interface names and their assigned IPs?  I am aware that I can do codeifconfig | grep inet/code but this does not include the interface name (en0, en1 etc)  <answer135166> You could run codeifconfig -l/code to get a list of interfaces, then iterate through them.  (bash)  precodefor i in `ifconfig -l `; do echo $i; ifconfig $i | grep "inet " ; done /code/pre  This will print the interface on one line, and the "inet " info on the next line.
