"Reveal" = "$@r"; "Minimize" = "\0"; "Minimize All" = "\0"; }'  defaults write com.apple.Safari NSUserKeyEquivalents '{ "Reopen Last Closed Window" = "~@r"; "Reopen All Windows from Last Session" = "~$@r"; "Mail Contents of This Page" = "\0"; }' /code/pre  The shortcut format is described in http://www.hcs.harvard.edu/~jrus/site/cocoa-text.html. If a menu item was assigned to nil, pressing L would trigger it in some applications like Audacity.
