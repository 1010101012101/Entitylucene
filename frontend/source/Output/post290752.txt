Programmatically reboot into Bootcamp on 10.11 and later <body> Prior to OSX 10.11, when SIP was introduced you could apparently (I never tried it) reboot into Boot camp programmatically like this:  precodesudo bless --mount "/Volumes/BOOTCAMP" --setBoot -nextonly sudo reboot /code/pre  However since SIP was added, you get this error:  precodeCould not set boot device property: 0xe00002bc /code/pre  It is possible to disable SIP by rebooting into recovery mode (hold Command-R on boot), and running codecsrutil disable/code in the terminal.  Unfortunately, bless still does not work.
