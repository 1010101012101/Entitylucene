How to get defined lines from OS X Terminal result? <body> How do I get the first 20 lines of a OS X Terminal result?  <answer238734> The command you want is head  so for a command x and the first 20 lines you enter  precodex | head -n 20 /code/pre  This runs x and pipes the input into the head command.  <answer238735> If the output is from a command you are running, you can pipe the output to the codehead/code command.  precode$ ls -l | head -n 10 /code/pre
