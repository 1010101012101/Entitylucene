How to get a list of Apple Time Zones <body> How would I get a listing of all the time zones that Apple has on macOS. For example:  ul liPacific Daylight Time/li liCentral European Standard Time/li lietc./li /ul  img src="https://i.stack.imgur.com/rwfz1.png" alt="enter image description here"  <answer296258> macOS uses the same TimeZone layout as FreeBSD.    Look in code/usr/share/zoneinfo/code for each time zone file.  precodels /usr/share/zoneinfo Africa          Australia       Etc             MET             WET America         CET             Europe          MST             posixrules Antarctica      CST6CDT         Factory         MST7MDT         zone.tab Arctic          EET             GMT             PST8PDT Asia            EST             HST             Pacific Atlantic        EST5EDT         Indian          SystemV /code/pre  Traverse the directories to drill down to the timezone you need.  Eg.  Eastern time would be:  code/usr/share/zoneinfo/America/NewYork/code  The file is a C library.  To get the emoutput/em of the file, you can use one of two commands - codezdump/code or codefile/code:  ul licodezdump /usr/share/zoneinfo/America/New_York/code/li licodefile /usr/share/zoneinfo/America/New_York/code/li /ul  <comment374308> How would you read the file in plain text? `$  vim /usr/share/zoneinfo/America/Los_Angeles` just gives me a bunch of gibberish @Allan.
