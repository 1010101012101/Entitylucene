It only reveals the already running one   How can I start a new "tab" or a new "iTerm window"?  <answer164684> You can create an AppleScript that emulates human entry manually. In other words, drag and drop AppleScript and force the application to become the frontmost application and then invoke a contextual menu click as if you clicked it with your mouse. Then re-save it as your service.   And example, but not specific to your application would be:  precodeactivate application "ApplicationName" tell application "System Events" tell process "ApplicationName"     keystroke "n" using {command down}     delay 1 end tell end tell /code/pre  <answer164697> You can use iTerm keyboard shortcut:  ul liNew Tab: kbdCommand ⌘/kbd+kbdT/kbd/li liNew Window: kbdCommand ⌘/kbd+kbdN/kbd/li /ul
