How to show/hide invisible files and folders from Service menu? <body> How to show/hide invisible files and folders from Service menu?  By default files and folders whose name start with . is invisible. In order to show them, you can use the following command in Terminal.  precodedefaults write com.apple.finder AppleShowAllFiles -boolean true; killall Finder /code/pre  To restore the default state, in which those files are hidden, use the following.  precodedefaults delete com.apple.finder AppleShowAllFiles; killall Finder /code/pre  How can I toggle the status of codeAppleShowAllFiles/code key without launching Terminal?  <answer181699> ol liLaunch strongAutomator/strong/li liChoose strongService/strong for new document./li liFrom "Actions" menu, choose strongUtilities/strong  strongRun Shell Script/strong.
