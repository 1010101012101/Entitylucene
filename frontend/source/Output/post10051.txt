Changing Chrome's User Agent? <body> Is it possible to change Chrome (or Chrome based browsers) user agents on OSX? I dont see a menu or preference option for doing so.  <answer10052> This link shows how to do it using Terminal. The Terminal command that opens Chrome using the Safari user agent string:  precodeopen /Applications/Google\ Chrome.app   --args -user-agent="Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10_5_3; en-us) AppleWebKit/525.19 (KHTML, like Gecko) Version/3.2.1 Safari/525.19" /code/pre  I tested this, and it works with the current version of Chrome.
