Get Number of Screens Using system_profiler <body> Using the codesystem_profiler SPDisplaysDataType/code command in terminal will give info of all connected displays; something like this.  precodeGraphics/Displays:      NVIDIA GeForce GT 640M:        ...        Displays:         iMac:           ...         DELL P2214H:           ...         DELL P2214H:           ... /code/pre  Is it possible to get the total number of connected displays, in this case 3? If not, then how can the above output be parsed to return the total connected displays.  <answer254924> This can be achieved by parsing the output of codesystem_profiler SPDisplaysDataType/code with this line.  codesystem_profiler SPDisplaysDataType | grep Resolution: | wc -l/code  As fd0 mentioned, this can be shortened to this.  codesystem_profiler SPDisplaysDataType | grep -c Resolution/code  <comment317453> Your solution can be shorten- `system_profiler SPDisplaysDataType | grep -c Resolution` <comment317456> @fd0 Thanks for this.
