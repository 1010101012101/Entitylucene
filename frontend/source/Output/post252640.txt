OSX tags from terminal/console <body> Is it possible to create some script to show all the finder-tagged items in terminal/shell?   I'd like to create something like this:  ls-tag "Music" to display all the items of Music tag  <answer252643> No script is needed to accomplish that. Simply use codemdfind 'kMDItemUserTags=&lt;tag&gt;'/codeto find all items with the tag code&lt;tag&gt;/code.  In your case that's codemdfind 'kMDItemUserTags=Music'/code.  If you want to restrict the search to a special directory use:  precodemdfind 'kMDItemUserTags=Music' -onlyin /path/to/folder /code/pre  e.g.  precodemdfind 'kMDItemUserTags=Music' -onlyin ~/Music/Classical\ Music/Сергей\ Сергеевич\ Прокофьев /code/pre  <answer252649> There is a commandline tool called strongtag/strong:  tag is a command line tool to manipulate tags on Mac OS X files (10.9 Mavericks and above), and to query for files with those tags. tag can use the file system's built-in metadata search functionality to rapidly find all files that have been tagged with a given set of tags.
