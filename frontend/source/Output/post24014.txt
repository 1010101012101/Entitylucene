Specifically the BT Analog Chorus CH2S-3 plugin from Nomad Factory. The Nomad Factory FAQ suggests the following procedure if one of their plugins is crashing Logic:     Please follow the steps below to resolve current issues:      ol   liDownload our latest installer from our downloads page./li   liDouble-Click the "uninstall_AU.command" included in the "Support   OpenMe/Uninstall Your Software" folder to remove the installed   plug-ins./li   liDouble-Click the "backup_licenses.command" included in the   "Support OpenMe/Backup Your Licenses" folder, the Nomad Factory folder   will be renamed to "Nomad Factory Backup", Presets and Licenses will   be saved./li   liWhen ready, run the installer. This will create a new Nomad   Factory folder including the Licenses and Plug-In Settings folders   with the correct permissions settings. (You can now drag any backed up   licenses/presets from the backup folder directly into the newly   created folders)      ul   liThe Backup folder can be found here : code/Library/Application   Support/Nomad/code Factory Backup/li   liThe new created Nomad Factory folder can be found here :   code/Library/Application Support/Nomad Factory/code/li   /ul/li   /ol      After moving your saved licenses, launch your host application and it   should work. You can now delete the codeNomad Factory Backup/code folder.   If that doesn't work try removing this plugin from codeMacintosh HD/Library/Audio/Plug-Ins/Components/code and then starting Logic. You don't need to delete the file, just grab it and move it some place else like your Desktop. The file should be named something like codeMacintosh HD/Library/Audio/Plug-Ins/Components/BT Analog Chorus CH2S-3.component/code. As far as Logic is concerned, if there's no code.component/code file in a sub-directory of codeMacintosh HD/Library/Audio/Plug-Ins/Components/code then the plugin doesn't exist and it doesn't get loaded on boot up of Logic.  If removing the file from the codeComponents/code directory solves the problem then you've definitely pinpointed the issue. You can try and re-install your Nomad Factory BT Analog Chorus CH2S-3 plugin from the media you got when you purchased it and see if that fixes the problem. If that doesn't help, contact Nomad Factory support and let them know about your issues and that you tried their repair procedure already.  <comment26847> The times logic has crashed on me is usually due to plugins.
