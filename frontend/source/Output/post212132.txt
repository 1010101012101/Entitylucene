Adding Bookmarks for multiple tabs in Safari with a keyboard shortcut <body> In Safari, there is the codeAdd Bookmarks for These N Tabs…/code function. Unfortunately, you cannot create a Keyboard shortcut for that because those need to match the name of the menu item.  <answer212136> You can use this Applescript; it creates a shell script containing the links for all the tabs in the current Safari window:  precode-- Set the default folder to Home -&gt; bookmarks set bookmarkFolder to "/bookmarks" set username to do shell script "whoami" set defaultFolder to POSIX file ("/Users/" &amp; username &amp; bookmarkFolder)  -- Initialize the text ot the script. set cmd to "#!/bin/bash" &amp; linefeed &amp; linefeed  -- Add commands to open all the tabs. tell application "Safari"     set n to count of tabs in front window     repeat with i from 1 to n         set cmd to cmd &amp; "open -g " &amp; URL of tab i of front window &amp; linefeed     end repeat end tell  -- Open/create a file and save the script.  tell me     activate     set scriptAlias to choose file name default name "tabset" default location (defaultFolder as alias) end tell set scriptPath to quoted form of POSIX path of scriptAlias set scriptFile to open for access scriptAlias with write permission set eof scriptFile to 0 write cmd to scriptFile starting at eof close access scriptFile  -- Change the file attributes to make it double-clickable. do shell script "chmod 777 " &amp; scriptPath do shell script "xattr -wx com.apple.FinderInfo '00 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00  00 00 00 00 00 00 00 00  00 00 00 00 00 00 00 00' " &amp; scriptPath /code/pre  The original version is by Dr. Drang.
