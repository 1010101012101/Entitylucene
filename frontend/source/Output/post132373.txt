Adding a username alias using the command line <body> How do I add a username alias using the command line?  I can do it fine using the preferences dialog through the "Advanced Options", but I need to do it during an installation script.   Can someone point me the right way?  <answer132384> The information is accessible through codedscl/code. To add the alias "visitor" and be able to use this username for guest login, use the following command:  codesudo dscl . -merge Users/Guest RecordName visitor/code  <comment155654> See this link: http://serverfault.com/questions/102697/unix-user-alias
