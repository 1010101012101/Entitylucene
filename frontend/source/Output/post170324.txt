If the problem persists get hold of a USB hard disk and install a clean copy of OS X on it, then see if you get exactly the same issue when the system goes to sleep.  If you do have the same problem it will point to a hardware issue, possibly a GPU re-ball/re-flow (a pain but ultimately a lot cheaper than buying a new or even secondhand MacBook).
