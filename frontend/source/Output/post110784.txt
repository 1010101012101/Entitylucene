Install Flash Player 11.9 on case-sensitive filesystem <body> I'm trying to install Flash Player 11.9 on Mac OS X running on a case-sensitive filesystem. The installation fails with a (very unsatisfactory) error message.  What can I do to complete the installation successfully?  <answer110785> As I posted on an Adobe forum post the reason is the following: The installation tries to access code/usr/libexec/Plistbuddy/code but the correct spelling is code/usr/libexec/PlistBuddy/code. You can execute the following commands in a terminal to solve the problem and successfully complete the installation:  precodecd /usr/libexec sudo ln -s PlistBuddy Plistbuddy /code/pre
