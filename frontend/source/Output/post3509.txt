Can I set OS X to automatically open ICS files in iCal? <body> Can I configure OS X so that the default open action on an ICS files is to open it in iCal?  <answer3511> If you highlight the file in the finder, press command-i to open the file info dialog. Change the entry "Open with" to "iCal" (select "other" and browse for iCal if it is not listed). Press the button "change all" to make "open in iCal" the default action.  <answer3515> Take a look at this Apple StackExchange question and answer.
