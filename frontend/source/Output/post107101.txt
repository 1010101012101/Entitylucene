How to remove the sleep button from the login window? <body> I'm trying to remove the "Sleep" button from the login window on some of my Mac's to prevent inadvertently disabling access to the remote server.   How to disable the sleep button from Mac OS X login screen?  <answer107102> Use the Terminal command:  precode$ sudo defaults write /Library/Preferences/com.apple.loginwindow SleepDisabled -bool True /code/pre  To re-enable the Sleep button later on, use:  precode$ sudo defaults write /Library/Preferences/com.apple.loginwindow SleepDisabled -bool False /code/pre
