Open iTerm and execute some commands automatically <body> I need to open some terminal sessions via iTerm 2 and execute commands on each terminal (e.g. rails s ; rails c)  How to do it via automator or alfred?  <answer158051> You could do it with an Alfred Worflow (you need the Powerpack).  First create a codeBlank Workflow/code and fill the data:  img src="https://i.stack.imgur.com/IqcfR.png" alt="enter image description here"  Use the plus symbol in the right cornet to add codeTrigger → Hotkey/code action. Define your prefered hotkey and save it.  img src="https://i.stack.imgur.com/R9ZHJ.png" alt="enter image description here"  Add codeActions → Run script/code. Use code/usr/bin/osascript/code as language. Paste the following script and save it. Change codels -l/code to whatever command you want to execute.  precodetell application "iTerm"     activate     set the clipboard to "ls -l"     delay 0.5     tell application "System Events"         keystroke "v" using command down         keystroke return     end tell end tell /code/pre  img src="https://i.stack.imgur.com/HRGrn.png" alt="enter image description here"  Connect the two actions and you will be ready.   img src="https://i.stack.imgur.com/m9605.png" alt="enter image description here"  img src="https://i.stack.imgur.com/0kn0e.png" alt="enter image description here"  Type your hotkey and it will open iTerm and execute your command.
