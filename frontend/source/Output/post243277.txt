How can I create an alias for Finder in OS X El Capitan? <body> I want to add a Finder shortcut to my Launchpad, but I am unable to do it.  Is there any way to create an alias for Finder in OS X El Capitan?  <answer243280> Run the following command in Terminal (it will create a symbolic link for all users):  precodesudo ln -s /System/Library/CoreServices/Finder.app /Applications /code/pre  Or just for currently logged on user:  precodeln -s /System/Library/CoreServices/Finder.app ~/Applications /code/pre  Finder will appear in the applications list opened with Launchpad.
