In which file Terminal.app stores its windows? <body> Usually after reboot/crash all Terminal windows are re-open automatically, this time they weren't.  Where (which folder/file) Terminal.app stores the session/information about its windows which suppose to be re-open?  <answer178286> It is stored in the user's Saved Application State folder, located at:  code/Users/$USER/Library/Saved Application State/com.apple.Terminal.savedState/code
