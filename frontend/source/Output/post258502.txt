How do I format a partition with FAT32? <body> The command codediskutil eraseDisk/code is not what I want because that wipes the entire disk. I have a partition already allocated and it is blank. All I need to do is format it with VFAT/FAT32. How can this be done from Terminal? It must the same effect as using codemkfs.vfat/code on Linux.  <answer258503> The command is codenewfs_msdos/code.  Also for HFS+, the command is codenewfs_hfs/code.  Use codeman newfs_msdos/code or codeman newfs_hfs/code for more information.  Generally, you would use codediskutil list/code to get the slice (identifier).  Next you use codediskutil unmount identifier/code, followed by the format command.  Finally, use codediskutil mount identifier/code.  For example.  precodediskutil list diskutil unmount /dev/disk0s4 newfs_msdos -F 32 -v BOOTCAMP /dev/disk0s4 diskutil mount /dev/disk0s4 /code/pre  h2Note: Using codediskutil/code to format a partition (slice) can have the side effect of altering the type of partitioning a drive is using./h2  For example, the command  precodediskutil eraseVolume fat32 BOOTCAMP /dev/disk0s4 /code/pre  can result in changing a GPT partitioned disk into a hybrid GPT/MBR partitioned disk.  <answer258504> Enter codediskutil list/code to get the disk identifier/partition. Then enter:  precodediskutil eraseVolume FAT32 DATA diskXsY #replace diskXsY by the proper disk identifier /code/pre  emDATA/em will be the name of the volume.
