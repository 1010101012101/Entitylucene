How can I add the question mark via voice?  <answer29565> http://howto.cnet.com/how-to-make-siri-use-punctuation/8301-11310_39-20124379-285.html  To get Siri to use punctuation, you only need to speak the punctuation during the composition of your message. For example:     Hi comma how are you question mark did you see the game last night   exclamation point question mark.   Speaking that exact line to Siri would equate to the following message:     Hi, how are you?
