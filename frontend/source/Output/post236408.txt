How to remove the downloaded OS X El Capitan from App Store <body> I downloaded OS X El Capitan from the app store to create a bootable usb drive and now that I am done, I dont know how to delete it. There is no un-install option in the app store.   How can I delete the downloaded OS X from my macbook?  <answer236409> It's actually very easy....  h2Using Terminal/h2  Type the following command:  coderm -rf /Applications/Install\ OS\ X\ El\ Capitan.app//code  This will remove the the install and sub-folders for the El Capitan installer.  h2Using Finder/h2  Open the Applications Folder by navigating to it or by pressing kbdShift/kbdkbdCommand/kbdkbdA/kbd.  Select the emInstall OS X App/em and kbdControl/kbd Click or right click and select emMove to Trash/em  img src="https://i.stack.imgur.com/rmPhy.png" alt="enter image description here"  That's it.  The installer has been removed!
