Airport Utility - The code execution cannot proceed because dnssd.dll was not found. Reinstalling the program may fix the problem <body> After installing Airport Utility for Windows I received this error.     The code execution cannot proceed because dnssd.dll was not found.   Reinstalling the program may fix the problem   <answer290751> Bonjour is likely to be missing. Bonjour on Windows is bundled with Bonjour Print Services for Windows  After installing that, Airport Utility should run straight away.  This is what worked for me.
