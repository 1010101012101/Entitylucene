If you are certain that Stealth Mode is disabled I would launch a packet capture with Wireshark and see if you get anything back from your ping6.  If you have an other computer I would try to ping6 :  Mac WiFi &lt;= router &lt;= PC Ethernet  Mac WiFi &lt;= router &lt;= PC WiFi  PC WiFi &lt;= router &lt;= Mac WiFi  PC WiFi &lt;= router &lt;= Mac Ethernet  Mac Ethernet &lt;= router &lt;= PC Ethernet  It would help you to figure is Mac is not able to craft/receive ping6 packets over WiFi, or if the issue is located to the router. You can also check the router firewall logs to see wether the firewall block those probes.
