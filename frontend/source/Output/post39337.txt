How can I run a hardware test in lion? <body> My Mac was shipped with Lion. Therefore I have no install CDs that I could use to run a hardware test.  How can I run a hardware test?  <answer39338> If your mac shipped with Lion, you should be able to run the hardware test by simply holding kbdD/kbd when your computer boots.
