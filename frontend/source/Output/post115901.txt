How to change default font size when preview open microsoft excel document? <body> I use macbook air, Preview is brilliant, But seems can not zoom out microsoft document(word, excel), So I want to change the default font size.  But how?  <answer152673> Enter the following command into the terminal. Quit and relaunch Preview.  precodedefaults write com.apple.Preview NSSystemFontSize -int X /code/pre  Change X to desired font size.
